# RECONSTRUCCION 3D

Reconstruccion 3D estereográfica

## Requerimientos

OpenCV 2.4.10
OpenGL

## Ejecución (con Zinjai)

### Windows

- Instalar [Zinjai](http://prdownloads.sourceforge.net/zinjai/zinjai-w32-20161214.exe?download) con OpenGL incluido
- Instalar complemento OpenCV 2.4.10 [Windows](https://sourceforge.net/projects/zinjai/files/complements/zinjai-add-opencv2-w32-20150320.zcp/download)

### Linux (Ubuntu)

- Instalar OpenCV 2.4.10
- Instalar OpenGL
- Instalar [Zinjai](http://prdownloads.sourceforge.net/zinjai/zinjai-l64-20161214.tgz?download)
- Instalar complemento OpenCV 2.4.10 [Linux](https://sourceforge.net/projects/zinjai/files/complements/zinjai-add-opencv2-lnx-20150320.zcp/download), después reiniciar Zinjai
